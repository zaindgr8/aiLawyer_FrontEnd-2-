Troubleshooting

If you cannot connect to the backend, check the following:

  1. Is your Python FastAPI backend running?
  2. Is it running on port 8000? If not, you need to update the API_URL in AppContext.js
  3. Does your backend have CORS enabled? Your FastAPI code should include CORS middleware.

Example FastAPI CORS Setup:

from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI()

# Add CORS middleware
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # For development only! Restrict this in production
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.post("/api/chat")
async def chat(request: dict):
    # Your chat logic here
    return {"response": "This is a test response", "thread_id": "test123"}
